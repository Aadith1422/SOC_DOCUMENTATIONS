# SOC Documentation Repository

This repository contains Security Operations Center (SOC) incident investigation reports.

## Structure

- Incident_1 – SharePoint ToolShell RCE investigation
- Incident_2 – Web attack analysis
- Incident_3 – Post-exploitation activity review

Each incident includes:
- Alert overview
- Timeline
- Evidence screenshots
- Technical analysis
- MITRE ATT&CK mapping
- Remediation steps

> All data is sanitized and created for learning and portfolio purposes.
